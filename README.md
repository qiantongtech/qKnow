# qKnow

#### 介绍
千知平台是一个面向企业级应用的开源知识管理系统。平台深度融合了知识抽取、知识融合、知识推理与知识图谱等核心功能模块，支持从非结构化和结构化数据中高效提取知识，助力企业快速构建可落地的知识图谱体系。
